Make money In-text advertising solution for WordPress users, automatically links high value keywords in your posts to searches on ebay.

== Description ==
WP ZinText is an easy way to implement in-text advertising on your blog. Link ads are only displayed on single post pages, and you can set them to only display on posts older than X days.

Question / Problems with WP ZinText? Go Here: http://www.zintext.com

To use this plugin you will need a campaign ID from eBay Partner Network. For more information on ePN and what a campaign ID is go here: http://www.zintext.com/faq.php


== Installation == 

This section describes how to install the plugin and get it working.

1. Upload `wpzintext.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the Settings -> wpZintext link
4. Follow the easy instructions on the settings page to get going

In order to earn money with this plugin you will need to register at [eBayPartnerNetwork.com](http://www.ebaypartnernetwork.com) and create a campaign ID.

= Where do I get a eBay Campaign ID? = 

You will need to register at the [eBay Partner Network](https://www.ebaypartnernetwork.com) and create a campaign id. A Campaign ID is issued to you by eBay Partner Network. To find your Campaign ID, log into the eBay Partner Network and click "Campaigns" in the top menu bar. You will be given a list of your current campaigs and their IDs, simply copy and paste the one you want to use or create a new one.
